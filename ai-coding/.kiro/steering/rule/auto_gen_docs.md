# **文档更新要求**
**重要：** 在完成任何导致代码库更改的任务后，您必须创建一个任务来更新相关文档：
## **文档更新任务创建**
- **架构更改：** 如果任务修改了系统架构，请创建任务以更新 `gen/docs/architecture/` 文件
- **功能更改：** 如果任务修改了功能性，请创建任务以更新 `gen/docs/features/{{feature_name}}/` 文件
- **运营更改：** 如果任务影响部署或运营，请创建任务以更新 `gen/docs/operations/` 文件
- **模式更改：** 如果任务修改了数据模型或模式，请创建任务以更新 `gen/docs/schemas/` 文件
## **必需的文档更新**
每个文档更新任务必须指定：
- 需要更新的具体文档文件
- 需要修改的部分
- 这些更改应如何在文档中反映
- 是否需要新建文档文件

## **完成确认**

完成后，你必须输出：

```
✓ doc完毕:{files}
```
